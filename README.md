This project focuses on **data cleaning** and **exploratory data analysis (EDA)** of the Netflix Titles dataset using **SQL**.

---

## Dataset
The dataset includes information about Netflix movies and TV shows such as:
- Content type (Movie / TV Show)
- Release year
- Country
- Genre
- Rating
- Duration

## Data Cleaning
The following steps were applied to prepare the dataset for analysis:
- Removed records with missing critical values to avoid misleading results
- Standardized duration values for movies and TV shows
- Ensured consistent formatting across categorical columns
- Prepared analysis-ready fields for duration and content type

---

## Exploratory Data Analysis (EDA)
EDA was performed to understand content distribution and trends.

### Key Analysis Areas
- Overall content overview 
- Time-based analysis 
- Country-based content distribution
- Genre distribution across movies and TV shows
- Duration analysis for movies and TV series
- Rating and target audience analysis
